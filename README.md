# 🎓 Student Management System (PHP + MySQL)

A simple CRUD-based Student Management System built using PHP, MySQL, and Bootstrap.  
This project allows you to manage student records easily with a clean UI and authentication system.

---

## 🚀 Features

- 🔐 Login System (Admin)
- ➕ Add Student
- 📋 View Students List
- ✏️ Edit Student Details
- ❌ Delete Student
- 🔍 Search Students
- 🎨 Bootstrap 5 UI

---

## 🛠️ Tech Stack

- PHP (Core PHP)
- MySQL Database
- Bootstrap 5
- HTML/CSS
